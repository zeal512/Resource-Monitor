# Resource Monitor Web Application

## Overview
The Resource Monitor web application is designed to offer users an intuitive interface to monitor and comprehend critical system metrics in real-time. 
It provides insights into CPU usage, memory usage, disk usage, Swapped Memory metrics, network usage, and the top 10 processes by memory usage. 
Additionally, it offers visual representations of memory usage through pie charts.

## Installation
To run this application, ensure you have Java 17 installed on your system.

## Getting Started
Step 1: Load the project. 
Make sure jdk 17 is installed.
(https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html) 

Step 2: Execute the src/main/java/com/example/Metrics/DemoApplication.java
(Make sure port 8080 is available.)

Step 3: After the DemoApplication.java executes, open browser and hit below URL
http://localhost:8080/memory.html


## Usage
Once the application is running, access it through your web browser at `http://localhost:8080/memory.html`. 
Here, you can interact with the application interface to monitor system metrics and gain insights into memory usage.



## Development
This project is built using Java 17 and utilizes various libraries and frameworks for web development.
